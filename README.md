# 🚪 English Janala (English Window)

**English Janala** is a dedicated web platform designed to make English language learning accessible and interactive for Bengali speakers. The name "Janala" (Window) symbolizes opening new opportunities through language mastery.

[**Live Demo**](https://sadiarahmansmrity0.github.io/english-janala/)

---

## 🎯 Project Purpose

Learning a new language can be daunting. **English Janala** breaks down the barriers by providing a structured path for students and enthusiasts to improve their grammar, vocabulary, and conversational skills through a clean, distraction-free interface.

## ✨ Key Features

- **Categorized Lessons:** Organized modules covering basic to advanced grammar.
- **Vocabulary Builder:** Curated lists of essential words with meanings and usage examples.
- **Interactive UI:** A modern, "pixel-perfect" design that ensures a smooth learning experience across all devices.
- **Resource Hub:** Quick access to common phrases and everyday conversation starters.
- **Progress Tracking:** Easy-to-navigate layout to keep learners motivated.

## 🛠️ Built With

- **HTML5:** Semantic structure for better accessibility.
- **CSS3 & Tailwind CSS:** Custom styling and responsive layouts.
- **JavaScript (ES6+):** Dynamic content rendering and interactive elements.
- **Google Fonts:** Clean typography for readability.
